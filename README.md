## Description

ConRec is a concept-coverage recommender system that integrates:

- Ontology-based modeling using OWL and SWRL
- Personalized recommendations based on FSLSM learner profiles
- Optimization of learning object selection using the Set Covering Problem (SCP)
- A chatbot interface for interaction and feedback
- A gamification module for engagement

The system was tested in a real classroom setting and aims to support personalized and explainable learning experiences.
